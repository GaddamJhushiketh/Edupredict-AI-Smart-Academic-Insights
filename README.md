# Edupredict AI – Smart Academic Insights 📊🎓

Edupredict AI is an AI-powered academic analytics platform designed to analyze student data and predict academic performance using machine learning techniques. The system helps institutions identify at-risk students early and provides insights for improving academic outcomes.

This project integrates data analytics, machine learning, and visualization to support smarter educational decision-making.

---

# 🚀 Features

* 📊 **Student Performance Analytics**
* 🤖 **AI-Based Prediction Models**
* ⚠️ **Early Warning System for At-Risk Students**
* 📈 **Graphical Performance Insights**
* 🗄 **Structured Academic Database**
* 🔐 **Secure Data Handling**
* 📄 **Automated Academic Reports**

---

# ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/edupredict-ai.git
```

### 2️⃣ Navigate to project folder

```
cd edupredict-ai
```

### 3️⃣ Create virtual environment

```
python -m venv venv
```

### 4️⃣ Activate environment

Windows

```
venv\Scripts\activate
```

Linux / Mac

```
source venv/bin/activate
```

### 5️⃣ Install dependencies

```
pip install flask pandas numpy scikit-learn matplotlib
```

---

# ▶️ Running the Project

Initialize the database first:

```
python init_db.py
```

Start the application:

```
python app.py
```

# 📊 How the System Works

1. Academic and student data is stored in the database.
2. The analytics engine processes the dataset.
3. Machine learning models analyze student performance patterns.
4. Predictions and insights are generated.
5. Educators can identify students needing academic support.
